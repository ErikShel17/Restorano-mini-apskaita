# 🍳 Breakfast Order System

Paprasta C++ programa, skirta vietinio restorano pusryčių užsakymams ir sąskaitoms generuoti.

Programa leidžia vartotojui pasirinkti vieną arba kelis patiekalus iš meniu, įvesti norimą porcijų kiekį, apskaičiuoja bendrą kainą su **21% PVM** ir sugeneruoja sąskaitą faile `receipt.txt`.

---

## ✨ Funkcionalumas

- 📋 Meniu nuskaitomas iš išorinio failo `menu.txt`
- 🍽️ Vartotojas gali pasirinkti vieną arba kelis patiekalus vienu metu
- 🔢 Kiekvienam patiekalui galima nurodyti porcijų kiekį
- ✅ Programa patikrina, ar pasirinktas patiekalo numeris egzistuoja
- 💶 Apskaičiuojama suma be PVM
- 🧾 Apskaičiuojamas 21% PVM mokestis
- 🖥️ Galutinė sąskaita išvedama į ekraną
- 💾 Sąskaita išsaugoma faile `receipt.txt`

---

## 🛠️ Naudojamos technologijos

- 💻 C++
- 📖 Failų nuskaitymas su `ifstream`
- ✍️ Failų rašymas su `ofstream`
- 🧱 Struktūros
- 📦 Masyvai
- 🔁 Ciklai
- 🔀 Sąlygos
- 🎯 Formatavimas su `iomanip`
